== Description ==

LCPay payment gateway plugin for WooCommerce. This plugin enable online payment using credit or debit cards (Visa and Mastercard only) and online banking (for Malaysian banks only). Currently LCPay is only available for businesses that reside in Malaysia.
This plugin will use LCPay payment gateway engine to process transaction. 

Official Page : https://lcpay.my/

Term and Conditions : https://register.lcpay.my/terms


== Frequently Asked Questions ==

= Do I need to sign up with LCPay in order to use this plugin? =

Yes, we require info such as merchant id and secret key that is only available after you sign up with senangPay.

= Can I use this plugin without using WooCommerce? =

No.
